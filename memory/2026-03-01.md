## Greenville Dashboard Project

- Started building local dashboard for Greenville, TX
- Data sources:
  - Weather: Open-Meteo API (switched from wttr.in which was failing)
  - Jail: apps.huntcounty.net - jail roster with inmate names/dates
  - News: Herald Banner (heraldbanner.com)
  - Traffic/Map: OpenStreetMap
- Created HTML dashboard with weather, hourly forecast, radar placeholder, jail list, news
- Localtunnel used for remote access (sometimes unreliable)

## Jail Data Investigation

- Discovered eextra.news has inmate photos
- Hunt County Sheriff site (apps.huntcounty.net/jail) uses JavaScript forms
- Found booking page: booking.asp?partyID=XXXXX
- Booking page shows: mugshot (base64 encoded), charges, bond amount, court dates
- Site uses form POST, hard to scrape automatically
- **MUGSHOTS FOUND!** - They are embedded as base64 data in the booking.asp page
- PartyIDs extracted from page source: 497184, 1325531, etc.
- Can extract full inmate details + photos by fetching booking.asp?partyID=XXXXX
