
## Kalshi BTC Trader - Cron Setup

The `kalshi_trader.py` was running as a bare `nohup` background process. Machine was rebooted April 1 and the process never restarted. Added `--once` flag to allow cron-based execution:

```
if __name__ == "__main__":
    import argparse
    parser = argparse.ArgumentParser()
    parser.add_argument("--once", action="store_true")
    args = parser.parse_args()
    if args.once:
        run()
    else:
        while True: ...
```

Crontab command (Captain needs to run via `crontab -e` since no root):
```
*/5 * * * * cd /home/colton/.openclaw/workspace/kalshi && /usr/bin/python3 kalshi_trader.py --once >> trades.log 2>&1
```

The script handles market hours (7-19 ET) internally.
